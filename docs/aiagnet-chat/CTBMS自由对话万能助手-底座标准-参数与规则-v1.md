# CTBMS 自由对话万能助手底座标准-参数与规则 v1.0

## 1. 参数分层标准

参数分为三层：

1. 会话级（Session）：临时参数，默认短期有效。
2. 用户级（User）：偏好参数，经多次命中后沉淀。
3. 全局级（Global）：平台规则参数，需审批发布。

## 2. 覆盖优先级

1. 会话级 > 用户级 > 全局级。
2. 冲突时必须记录冲突来源和消解结果。

## 3. 动态参数生成标准

1. LLM 可生成临时参数，但必须经过 schema 校验。
2. 临时参数默认 TTL（建议 24h）。
3. 临时参数不可直接升为全局参数。

## 4. 规则标准

1. 规则必须定义触发条件、动作、优先级。
2. 规则必须声明作用域与风险级别。
3. 高风险规则修改必须审批。

## 5. 验收标准

1. 参数冲突可解释。
2. 规则冲突有稳定消解顺序。
3. 参数/规则更新可审计可回滚。
