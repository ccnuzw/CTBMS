# CTBMS 自由对话万能助手底座标准-数据源 v1.0

## 1. 数据源接入必填项

1. `sourceCode` / `sourceName`
2. 访问协议与认证方式
3. 数据结构 Schema
4. 更新频率与新鲜度指标
5. 可用性 SLA
6. 权限域与脱敏策略

## 2. 数据质量标准

1. 完整性（字段缺失率）
2. 时效性（刷新延迟）
3. 一致性（多源对齐）
4. 可追溯性（来源与时间戳）

## 3. 调用标准

1. 查询必须带 traceId。
2. 超时后需触发降级策略（缓存/备用源/提示补充）。
3. 不允许把未脱敏字段直接返回用户层。

## 4. 验收标准

1. 数据源异常不会导致会话链路整体失败。
2. 降级路径可稳定执行。
3. 数据来源可在后台审计。
