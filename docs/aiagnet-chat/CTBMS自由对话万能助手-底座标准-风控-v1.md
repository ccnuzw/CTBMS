# CTBMS 自由对话万能助手底座标准-风控 v1.0

## 1. 风险分级

1. LOW：只读分析、低成本、无外部副作用。
2. MEDIUM：外发动作、有限副作用、可恢复。
3. HIGH：高成本、高权限、不可逆动作。

## 2. 执行策略

1. LOW：自动执行。
2. MEDIUM：提示后执行或一次轻确认。
3. HIGH：强确认或审批后执行。

## 3. 风控检查点

1. 能力调用前：权限、风险等级、预算。
2. 执行中：超时、重试、速率限制。
3. 执行后：审计记录、异常告警。

## 4. 必备能力

1. 请求幂等控制。
2. 预算与配额控制。
3. 敏感操作白名单。
4. 一键降级与回滚开关。

## 5. 验收标准

1. 高风险动作无绕过路径。
2. 失败动作可追踪可回滚。
3. 风控命中可解释。
