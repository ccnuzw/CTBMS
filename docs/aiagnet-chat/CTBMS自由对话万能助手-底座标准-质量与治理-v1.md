# CTBMS 自由对话万能助手底座标准-质量与治理 v1.0

## 1. 质量指标体系

### 1.1 体验指标

1. 首次结果时长（P95）
2. 用户追问次数
3. 会话完成率
4. 发送转化率

### 1.2 能力指标

1. 复用命中率
2. 临时能力创建率
3. 临时能力晋升率
4. 临时能力淘汰率

### 1.3 运行指标

1. 执行成功率
2. 平均时延
3. 平均成本
4. 回退触发率

## 2. 治理规则

1. 同类能力指纹去重，禁止重复膨胀。
2. 低质量临时能力自动淘汰。
3. 高复用高质量能力自动提案晋升。
4. 高风险能力必须经过审批闭环。

## 3. 审计要求

1. 路由决策可追踪。
2. 执行过程可回放。
3. 参数规则变更可溯源。

## 4. 发布门禁

1. `pnpm --filter api run type-check`
2. `pnpm --filter web run type-check`
3. `pnpm --filter api run test:e2e:agent-suite`
4. `pnpm agent:pre-release`

## 5. 验收标准

1. 关键指标可视化。
2. 治理动作可自动化执行。
3. 灰度与回滚演练可复现。
