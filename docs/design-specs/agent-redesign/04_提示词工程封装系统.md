## 1. 结构化模版引擎 (Template Engine)

我们使用 **Jinja2** (Python) 或 **Handlebars** (Node.js) 作为渲染引擎，Schema 严格遵循 Zod 定义。

### 1.1 Template Data Schema

```typescript
import { z } from 'zod';

const PromptTemplateSchema = z.object({
  meta: z.object({
    id: z.string(),
    description: z.string(),
    model_config: z.object({
      temperature: z.number().min(0).max(2).default(0.7),
      top_p: z.number().default(0.9),
      max_tokens: z.number().optional()
    })
  }),
  
  system: z.object({
    role_definition: z.string(), // "You are a..."
    constraints: z.array(z.string()), // ["No markdown", "Be concise"]
    style_guide: z.string().optional() // "Use professional tone"
  }),

  few_shot: z.object({
    enabled: z.boolean(),
    collection_id: z.string(), // Vector DB Collection
    k: z.number().default(3)   // Number of shots
  }),

  chain_of_thought: z.enum(['NONE', 'REACT', 'PLAN_SOLVE']).default('NONE')
});
```

## 2. 动态少样本注入 (Dynamic Few-Shot)

### 2.1 FewShotRetriever 核心类

```python
class FewShotRetriever:
    def __init__(self, vector_store, embed_model):
        self.store = vector_store
        self.embed = embed_model

    async def retrieve(self, task_description: str, k: int = 3) -> List[dict]:
        """
        1. Embed the current task description (User Query)
        2. Search in 'prompt_examples' collection
        3. Return top-k similar examples
        """
        query_vec = await self.embed.aembed_query(task_description)
        results = await self.store.search(
            collection_name="prompt_examples",
            vector=query_vec,
            limit=k,
            score_threshold=0.8
        )
        return [res.payload for res in results]
```

## 3. 思维链封装 (CoT Encapsulation)

### 3.1 ReAct 模式下的隐藏指令

前端不显示复杂的 "Thought/Action/Observation" 循环，只显示结果。为了实现这一点，必须在 System Prompt 中强制注入 Stop Sequences。

*   **Stop Sequences**: `["Observation:"]`
*   **Parsing Logic**:
    ```python
    def parse_stepped_response(llm_output: str):
        # Regex to capture Thought and Action
        pattern = r"Thought: (.*?)\nAction: (.*?)\nAction Input: (.*)"
        match = re.search(pattern, llm_output, re.DOTALL)
        if match:
             return {
                 "thought": match.group(1),
                 "tool": match.group(2),
                 "input": match.group(3)
             }
        return {"final_answer": llm_output}
    ```
