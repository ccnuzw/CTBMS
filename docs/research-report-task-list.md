# 商情中心 - 研究报告功能任务清单

> **模块位置**: 商情中心 → 智能采集 → 研究报告  
> **更新日期**: 2026-01-28

---

## 一、已交付核心功能 (✅ Completed)

### 1.1 全流程智能采集
- [x] **多模态支持**: 支持纯文本、PDF文档、图片上传采集
- [x] **智能分类引擎**: 自动识别报告内容类型 (`POLICY`/`MARKET`/`RESEARCH`/`INDUSTRY`)
- [x] **智能周期识别**: 自动提取报告发布周期 (`DAILY`/`WEEKLY`/`MONTHLY`/`QUARTERLY`/`ANNUAL`)
- [x] **AI 深度解析**: 自动提取标题、发布机构、核心观点、预测方向、关键数据点
- [x] **双向关联**: 采集情报与研报实体自动关联，支持数据回溯

### 1.2 独立研报管理系统
- [x] **研报列表页** (`/intel/research-reports`)
  - 支持多维度组合筛选 (类型/状态/品种/区域)
  - 集成全文检索 (标题/摘要/观点)
  - 支持批量操作 (批量删除/批量导出Excel)
  - 审核状态流转管理 (通过/驳回)
- [x] **研报详情页** (`/intel/research-reports/:id`)
  - 完整的元数据展示 (含阅读/下载统计)
  - AI 分析结果可视化 (观点情绪/预测逻辑/数据表格)
  - 文档在线预览与下载
  - 智能关联推荐 (同类研报)

### 1.3 研报统计驾驶舱
- [x] **仪表盘页面** (`/intel/research-reports/dashboard`)
  - 研报收录趋势分析 (按日/周/月)
  - 内容类型与来源机构分布图表
  - 实时更新动态流
  - 关键指标卡片 (总数/本周新增/阅读总量)

### 1.4 系统深度整合
- [x] **情报流融合**: B类情报流 (`IntelligenceFeed`) 无缝展示研报卡片
- [x] **双向溯源**: 
  - 情报流 -> 研报详情 (点击跳转)
  - 研报详情 -> 原始采集情报 (引用溯源)
- [x] **菜单集成**: 桌面端与移动端导航全覆盖

### 1.5 基础设施与质量保障
- [x] **数据模型**: 完善的 Prisma Schema 设计 (版本控制/审核状态/统计字段)
- [x] **类型安全**: 全链路 TypeScript 类型定义与检查通过
- [x] **测试数据**: 完整的 Seed 脚本覆盖各类研报场景

- [ ] **产业链图谱**: 在研报中自动标注产业链节点

### 2.3 工作流集成 (Priority: Low)
